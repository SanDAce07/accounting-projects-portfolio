# Templates

Reusable accounting, audit, and reconciliation templates for month-end close, AR review, bank reconciliation, workpaper documentation, and internal controls.

## Available Templates

| Template | Description |
| --- | --- |
| [Bank Reconciliation Template](./bank-reconciliation-template.md) | Month-end bank reconciliation with book-side and bank-side adjustment schedules, JE section, and sign-off |
| [AR Aging Review Template](./ar-aging-review-template.md) | Aging summary by bucket, customer concentration review, exception log, allowance assessment, and collection log |
| [Audit Workpaper Template](./audit-workpaper-template.md) | Full workpaper shell with objective, population, procedures, testing schedule, exception log, and conclusion |
| [Internal Controls Checklist](./internal-controls-checklist.md) | Controls checklist covering AR, cash, payroll, financial reporting, and ITGCs |
| [Month-End Close Checklist](./month-end-close-checklist.md) | Day-by-day close task checklist covering cutoff, reconciliations, accruals, flux review, and sign-off |

## How to Use

Each template is a standalone markdown file. Download or copy the file, fill in the header fields, and work through each section. All templates are designed to be:
- **Recruiter-friendly** — structured to show real accounting workflow knowledge
- **Audit-ready** — include the kind of documentation detail expected in a real close or workpaper environment
- **Practical** — based on real workflows used in staff accounting, AR review, and audit support roles
