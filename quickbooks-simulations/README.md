# QuickBooks Workflow Guides

Process documentation covering QuickBooks-style accounting workflows including AR cycle management, bookkeeping, payroll support, and month-end close. These guides support the completed Excel and audit projects; they are not presented as evidence of a completed QuickBooks company-file simulation.

## Projects

### AR Cycle Workflow
A walkthrough of the end-to-end AR workflow in QuickBooks, covering:

| Step | Action | QuickBooks Feature |
| --- | --- | --- |
| 1 | Set up customer records | Customer Center |
| 2 | Create and send invoices | Create Invoices |
| 3 | Record customer payments | Receive Payments |
| 4 | Apply payments to invoices | Apply to Invoice |
| 5 | Handle unapplied cash | Customer Credits |
| 6 | Run aging report | AR Aging Summary / Detail |
| 7 | Write off uncollectible accounts | Create Credit Memo + Bad Debt Item |

**Concepts covered:** invoicing, cash application, unapplied cash, aging review, write-offs, and credit memo processing.

---

### Month-End Close Workflow
A documented close sequence for a small business using QuickBooks, including:

1. Reconcile all bank and credit card accounts
2. Review open AR aging — investigate overdue items
3. Review open AP aging — confirm all liabilities recorded
4. Run and review Profit & Loss and Balance Sheet reports
5. Confirm payroll liabilities are posted and cleared
6. Lock prior period in QuickBooks (close date password)
7. File and archive supporting schedules

---

### Basic Bookkeeping Walkthrough
A bookkeeping exercise covering the full accounting cycle:
- Record daily transactions (sales, expenses, receipts)
- Post to general ledger accounts
- Run trial balance
- Make adjusting entries (accruals, depreciation)
- Generate adjusted trial balance
- Produce financial statements
